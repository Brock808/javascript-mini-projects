JavaScript Mini Projects
A collection of JavaScript mini‑projects showcasing my practice with backend fundamentals, data handling, and core programming logic. 
Each project focuses on concepts like arrays, objects, Maps, Sets, filtering, mapping, reducing, timers, and user input. 
This repository documents my progress as backend development.

Projects
1. Contact List Manager
Contacts stored as objects
Add/search/list functions
.filter() and .map()
Map for fast lookup

2. Event Countdown Timer
User inputs event name and date
Uses Date and setInterval
Calculates remaining time
Stops when the event is reached

3. Shopping Cart Simulator
Array of products
.filter() for price filtering
.map() for discounts
.reduce() for total cost
Set to avoid duplicates

4. Simple Quiz Game
Questions stored in objects
Loop with prompt()
Score tracking with an object method
Optional setTimeout for final score

5. Word Frequency Counter
Converts sentence to lowercase
Splits into words
Uses Map to count frequency
